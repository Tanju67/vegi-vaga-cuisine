## Built With

- React JS
- CSS Modules

## Getting Started

### Prerequisites

Install npm.

- npm
  ```
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo.
   ```
   git clone https://github.com/Tanju67/vegi-vaga-cuisine.git
   ```
2. Install NPM packages.

   ```
   npm install
   ```

3. Start the frontend server.
   ```
   npm start
   ```
